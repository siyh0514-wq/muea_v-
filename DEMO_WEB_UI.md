## 주요 기능

### 1. 클릭 선택
- ✅ 마우스로 카드 클릭
- ✅ 선택된 카드는 파란색 강조
- ✅ 재클릭하면 선택 취소

### 2. 시각적 피드백
- ✅ 호버시 카드 확대
- ✅ 선택시 그라데이션 배경
- ✅ 애니메이션 효과

### 3. 반응형
- ✅ 데스크톱
- ✅ 태블릿
- ✅ 모바일

## 사용 방법

```bash
# 웹 UI 실행
python main.py --web

# 브라우저 자동 열림
# http://localhost:5000
```
