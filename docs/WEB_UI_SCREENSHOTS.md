## 📱 반응형 디자인

### 데스크톱 (1200px 이상)
- 키워드 카드: 한 줄에 3개
- 제목 카드: 한 줄에 1개 (전체 너비)
- 최대 너비: 1200px

### 태블릿 (768px - 1199px)
- 키워드 카드: 한 줄에 2개
- 제목 카드: 한 줄에 1개
- 여백 축소

### 모바일 (768px 미만)
- 키워드 카드: 한 줄에 1개 (세로 스택)
- 제목 카드: 한 줄에 1개
- 터치 친화적 크기

## 🎯 사용 팁

### 1. 키워드 선택 전략
- **최소 3개 선택 권장**: 다양한 유형 조합
- **행동유도형 필수**: CPC가 가장 높음
- **롱테일 포함**: 경쟁 낮고 전환율 높음

### 2. 제목 선택 기준
- **CTR 90 이상**: 최고 성과 예상
- **훅킹 요소 확인**: 손실 회피 > 시간 절약 > 타겟 특정
- **타겟 매칭**: 콘텐츠 의도와 일치하는 제목

### 3. 빠른 작업
- 키워드 카드를 빠르게 클릭
- 실수하면 다시 클릭하여 취소
- 모든 선택 완료 후 한 번에 생성

## 🔧 문제 해결

### 화면이 제대로 표시되지 않는 경우
1. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
2. 강력 새로고침 (Ctrl+F5 또는 Cmd+Shift+R)
3. 다른 브라우저로 시도 (Chrome, Firefox, Safari, Edge)

### 클릭이 작동하지 않는 경우
1. JavaScript가 활성화되어 있는지 확인
2. 콘솔 에러 확인 (F12 → Console 탭)
3. 서버가 실행 중인지 확인 (http://localhost:5000)

### 스타일이 깨진 경우
1. `static/css/style.css` 파일 존재 확인
2. 파일 권한 확인
3. 서버 재시작
