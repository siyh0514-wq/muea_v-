# 빠른 시작 가이드

## 개요
이 시스템은 이미지 1장과 간단한 대본만으로 완성도 높은 YouTube 숏폼을 자동 생성합니다.

## 🚀 30초 요약
1. 이미지 준비 + 대본 작성
2. `input/` 폴더에 업로드
3. 6-10분 후 YouTube에 자동 업로드 완료!

---

## 사전 준비 (1회만)

### 1. API 키 준비
필수 API 키를 발급받으세요:
- ✅ **Gemini AI** (이미 보유)
- ✅ **D-ID** (이미 보유)
- ✅ **Google/YouTube API** (이미 보유)
- ✅ **Google Search Console** (이미 보유)

자세한 발급 방법: [API 설정 가이드](./API_SETUP_GUIDE.md)

### 2. 환경 설정
API 키를 설정하세요 (노코드 플랫폼 또는 .env 파일):

```bash
GEMINI_API_KEY=your-key
DID_API_KEY=your-key
YOUTUBE_CLIENT_ID=your-id
YOUTUBE_CLIENT_SECRET=your-secret
```

## 첫 번째 비디오 만들기

### Step 1: 이미지 준비 (1분)
제품 사진 또는 일반 이미지를 준비하세요.

**권장 사양**:
- 형식: JPG, PNG
- 해상도: 최소 1024x1024
- 용량: 10MB 이하
- 제품 사진의 경우: 정면 촬영, 깨끗한 배경

**파일명**: `my_first_video.jpg`

### Step 2: 대본 작성 (2분)
`input/scripts/my_first_video.json` 파일을 생성하세요:

```json
{
  "title": "AI 자동화로 월 100만원 버는 방법",
  "script_text": "여러분, 2024년 최고의 부업을 알려드릴게요! AI 자동화 도구를 활용하면 클릭 몇 번으로 월 100만원 이상 수익을 만들 수 있습니다. 첫 번째, YouTube 숏폼을 자동으로 생성하세요. 두 번째, 고수익 키워드를 활용하세요. 세 번째, 매일 꾸준히 업로드하세요. 이 방법으로 실제로 많은 분들이 성공하고 있어요. 지금 바로 시작해보세요! 구독과 좋아요 부탁드려요!",
  "duration": 30,
  "voice_id": "ko-KR-SunHiNeural",
  "category": "finance"
}
```

**템플릿**: `input/scripts/script_template.json` 참조  
**예시**: `input/scripts/examples.json` 참조

### Step 3: 파일 업로드
두 파일을 `input/` 폴더에 업로드:
- `input/images/my_first_video.jpg`
- `input/scripts/my_first_video.json`

⚠️ **중요**: 파일명이 동일해야 합니다!

### Step 4: 자동 처리 시작
시스템이 자동으로:

1. **이미지 분석** (3초)
   - Gemini가 이미지 분석
   - 제품인지 일반 이미지인지 판단

2. **제품 리서치** (15초, 제품인 경우)
   - 제품명 추출
   - 판매처 검색
   - 가격 정보 수집

3. **키워드 최적화** (10초)
   - 고수익 키워드 추출
   - 최상단 노출 키워드 조합
   - 제목 자동 최적화
   - 해시태그 15개 생성

4. **썸네일 생성** (5초)
   - 텍스트 오버레이 추가
   - 9:16 비율 최적화

5. **비디오 생성** (5-8분)
   - D-ID로 말하는 비디오 생성
   - 한국어 음성 적용

6. **YouTube 업로드** (1분)
   - 자동 업로드
   - 최적화된 메타데이터 적용

### Step 5: 결과 확인 (10분 후)
완료되면 다음 파일들이 생성됩니다:
- `output/videos/` - 완성된 비디오
- `output/thumbnails/` - 썸네일
- `output/reports/` - 상세 리포트
- 이메일 알림 (YouTube 링크 포함)

## 대본 작성 팁

### 기본 구조 (30초 기준)
```
[0-3초] 후킹: "여러분, 이거 아세요?"
[3-25초] 본문: 핵심 메시지 3-5개
[25-30초] CTA: "구독과 좋아요 부탁드려요!"
```

### 음성 선택
- **밝고 친근한 톤**: `ko-KR-SunHiNeural` (여성) - 추천
- **전문적인 톤**: `ko-KR-InJoonNeural` (남성)
- **뉴스 스타일**: `ko-KR-BongJinNeural` (남성)

### 카테고리 선택
- `finance`: 재테크, 투자, 부동산
- `tech`: AI, 자동화, IT
- `business`: 창업, 마케팅
- `lifestyle`: 자기계발, 건강, 루틴
- `trending`: 트렌드, 바이럴

## 제품 리뷰 콘텐츠 만들기

### 제품 사진 준비
- 제품 정면 사진
- 깨끗한 배경
- 브랜드/제품명이 보이게

### 대본 예시
```json
{
  "title": "에어팟 프로 2 리뷰",
  "script_text": "에어팟 프로 2세대를 1주일 사용해봤어요. 첫 번째, 노이즈 캔슬링이 2배 좋아졌어요. 두 번째, 배터리가 6시간으로 늘었습니다. 세 번째, 공간 음향이 진짜 영화관 같아요. 쿠팡에서 329,000원에 구매했는데 만족도 100%! 링크는 설명란에 있어요!",
  "duration": 25,
  "voice_id": "ko-KR-SunHiNeural",
  "category": "tech"
}
```

### 자동 처리 결과
시스템이 자동으로:
- ✅ 제품명 인식: "Apple AirPods Pro 2세대"
- ✅ 최저가 검색: 쿠팡 329,000원, 네이버 335,000원
- ✅ 최적화된 제목: "에어팟 프로 2세대 솔직 리뷰 | 329,000원 최저가 발견 | 1분만에 보는 장단점"
- ✅ 제휴 링크 추가 가능

## 자주 묻는 질문 (FAQ)

### Q: 하루에 몇 개나 만들 수 있나요?
A: 기술적으로는 무제한이지만, YouTube 무료 할당량은 일일 6개입니다.

### Q: 비용은 얼마나 드나요?
A: 영상 1개당 약 $0.086 (약 100원)입니다.

### Q: 제품이 인식 안 되면요?
A: 일반 콘텐츠로 자동 전환됩니다. 문제없어요!

### Q: 대본을 길게 쓰면 어떻게 되나요?
A: 60초까지 가능하지만, 숏폼은 30초 권장입니다.

### Q: 영어 대본도 되나요?
A: 네, D-ID는 40개 이상 언어를 지원합니다. `voice_id`만 변경하세요.

### Q: 제품 사진이 아닌데 리서치가 실행되나요?
A: 아니요. AI가 자동으로 판단하여 제품이 아니면 건너뜁니다.

### Q: 썸네일을 직접 만들고 싶어요.
A: 가능합니다. `thumbnail_text`를 비워두면 자동 생성을 건너뜁니다.

## 다음 단계

### 배치 처리
여러 개를 한번에 처리하고 싶다면:
1. 여러 이미지와 대본을 준비

`output/reports/`의 리포트를 확인하여:
- 어떤 키워드가 효과적인지
- 어떤 카테고리가 잘 되는지
- 개선할 점은 무엇인지

### 최적화
성과가 좋은 패턴을 발견하면:
- 유사한 콘텐츠 더 만들기
- 키워드 데이터베이스 업데이트
- A/B 테스트
