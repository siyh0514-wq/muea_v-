# 🎯 고수익 키워드 선택 시스템

## 개요

애드센스/블로그 전문 컨설턴트의 노하우를 AI로 구현한 **키워드 및 제목 추천 시스템**입니다.

사용자가 주제만 입력하면:
1. ✅ **고수익 키워드** 자동 추출
2. ✅ **클릭률 높은 제목** 3개 추천
3. ✅ **인터랙티브 선택** UI로 원하는 키워드/제목 선택
4. ✅ **YouTube 숏폼 스크립트** 자동 생성

## 🚀 사용 방법

### 빠른 시작

```bash
# 키워드 선택 모드로 실행
python main.py --keyword

# 또는 직접 실행
python keyword_selector.py
```

### 사용 흐름

```
1. 주제 입력
   예: "청년도약계좌", "전기차 보조금", "근로장려금"
   ↓
2. AI 분석 (Gemini)
   - 핵심 니즈 파악
   - 고수익 키워드 추출
   - 검색 의도 매칭
   ↓
3. 결과 표시
   - 고수익 키워드 목록 (선택 가능)
   - 롱테일 키워드
   - 추천 제목 3개 (CTR 점수 포함)
   ↓
4. 사용자 선택
   - 원하는 키워드 번호 입력 (예: 1,3,5)
   - 원하는 제목 번호 입력 (예: 2)
   ↓
5. 자동 스크립트 생성
   - YouTube 숏폼용 대본 자동 작성
   - input/scripts/에 JSON 파일 저장
   ↓
6. 비디오 생성
   - 이미지 준비 후 python main.py 실행
```

## 📊 분석 기능

### 1. 핵심 니즈 파악
사용자가 검색하는 이유 분석:
- 환급 받고 싶음
- 신청 방법 알고 싶음
- 자격 조건 확인하고 싶음
- 비용/가격 알고 싶음

### 2. 3대 수익화 키워드 유형

#### 행동 유도형 (CPC 높음)
```
"신청방법", "다운로드", "바로가기", "조회", "확인"
```
→ 사용자가 즉시 행동하게 만드는 키워드

#### 고연령 타겟형 (CPM 높음)
```
40-60대가 많이 검색하는 키워드
→ 구매력 높은 연령층
```

#### 금융/돈 관련 (CPC/CPM 모두 높음)
```
"환급", "지원금", "할인", "혜택", "보조금"
```

### 3. 롱테일 키워드
경쟁이 적으면서 수요가 있는 세부 키워드:
```
"청년도약계좌 신청 바로가기"
"청년도약계좌 대상자 확인"
"청년도약계좌 지급일"
```

### 4. 클릭률 높은 제목 공식

```
[손실 회피] + [구체적 혜택] + [행동 유도]
```

**예시:**
- ✅ "놓치면 손해! 청년도약계좌 꼭 확인하세요"
- ✅ "청년도약계좌 신청방법 총정리 | 5분만에 완료"
- ✅ "대상자라면 꼭 보세요 | 청년도약계좌 신청 가이드"

## 💡 최적화 팁

### 1. 키워드 선택 전략

**다음 조합이 최고:**
```
[행동유도형] + [금융/돈관련] + [롱테일] 1-2개
```

**예시:**
- ✅ 신청방법 (행동유도형)
- ✅ 환급 (금융/돈관련)
- ✅ 신청 바로가기 (롱테일)

### 2. 제목 선택 기준

**CTR 점수 85 이상** 추천
- 90점 이상: 매우 우수
- 85-89점: 우수
- 80-84점: 보통

**훅킹 요소 확인:**
- 손실 회피: "놓치면 손해"
- 시간 절약: "5분만에"
- 타겟 특정: "대상자라면"

### 3. 카테고리별 최적 키워드

#### 재테크/금융
```
환급, 지원금, 할인, 혜택, 보조금, 세금, 공제
```

#### 정부 정책
```
신청방법, 자격조건, 대상자, 조회, 확인
```

#### 생활 정보
```
꿀팁, 방법, 가이드, 총정리, 완벽정리
```

## 🎯 성과 예측

### 키워드 품질에 따른 예상 성과

| 키워드 유형 | CPC | CTR | 예상 수익 |
|------------|-----|-----|----------|
| 행동유도형 | 높음 | 높음 | $20-50 |
| 금융/돈관련 | 매우높음 | 중간 | $30-80 |
| 고연령타겟 | 높음 | 중간 | $25-60 |
| 롱테일 | 중간 | 낮음 | $10-30 |

### 최적 조합 사용시

**키워드 3개 선택:**
- 행동유도형 1개
- 금융/돈관련 1개
- 롱테일 1개

**예상 성과:**
- CTR: 8-15%
- 조회수: 20,000-100,000
- 수익: $50-150/영상
