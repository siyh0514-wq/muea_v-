# 📄 YouTube 설명란 자동 생성 가이드

## 🎯 설명란이란?

YouTube 영상을 업로드할 때 **제목 아래에 표시되는 텍스트**입니다.
- 제품 정보
- 구매 링크
- 해시태그
- 구독 요청

등이 포함되어 **조회수와 수익을 높이는 중요한 부분**입니다!

---

## ✨ 자동 생성되는 설명란

이미지를 넣으면 **각 언어별로 자동으로 최적화된 설명란**이 만들어집니다.

### 📝 설명란 구성

#### 1. 제목 및 소개
```
📱 [자동 생성된 최적 제목]

💡 [제품 설명]
```

#### 2. 핵심 키워드
```
🔍 핵심 키워드: 최저가, 쿠폰, 할인코드
```
→ SEO 최적화로 검색 노출 상위!

#### 3. 구매 링크
```
🛒 구매 링크:
[여기에 제휴 링크를 붙여넣으세요]
```
→ 제휴 수수료 수익!

#### 4. 행동 유도 (CTA)
```
👍 도움이 되셨다면 좋아요와 구독 부탁드려요!
💬 궁금한 점은 댓글로 남겨주세요!
```
→ 참여율 향상!

#### 5. 해시태그
```
#최저가 #쿠폰 #할인코드 #숏폼 #쇼핑 #리뷰
```
→ 추가 노출 채널!

#### 6. 영상 정보
```
⏱️ 영상 길이: 약 25초
```

#### 7. 채널 홍보
```
━━━━━━━━━━━━━━━━━
📺 이 채널을 구독하고 최신 정보를 받아보세요!
━━━━━━━━━━━━━━━━━
```

## 📁 설명란 파일 위치

자동 생성 후 설명란은 다음 위치에 저장됩니다:

```
output/videos/
  ├── airpods_pro_v1_HD.mp4      # 10초 버전 영상
  ├── airpods_pro_v1_HD.txt      # 10초 버전 설명란 ⭐
  ├── airpods_pro_v2_HD.mp4      # 25초 버전 영상
  ├── airpods_pro_v2_HD.txt      # 25초 버전 설명란 ⭐
  ├── airpods_pro_v3_HD.mp4      # 50초 버전 영상
  └── airpods_pro_v3_HD.txt      # 50초 버전 설명란 ⭐
```

**`.txt` 파일을 열어서 복사하여 YouTube에 붙여넣으면 됩니다!**

## 🎬 YouTube 업로드 방법

### 1단계: 영상 업로드
YouTube Studio → 만들기 → 동영상 업로드

### 2단계: 제목 입력
자동 생성된 제목을 복사하여 붙여넣기

### 3단계: 설명 입력
`.txt` 파일 내용을 **전체 복사**하여 붙여넣기

### 4단계: 구매 링크 추가
"[링크를 여기에 삽입하세요]" 부분에 실제 제휴 링크 입력

### 5단계: 썸네일 업로드
`output/thumbnails/` 폴더의 썸네일 선택

### 6단계: 게시!

## 🔧 설명란 수정 방법

자동 생성된 설명이 마음에 안들면 `.txt` 파일을 직접 수정하세요!

### Windows
```
notepad output/videos/airpods_pro_v2_HD.txt
```

### Mac
```
open -a TextEdit output/videos/airpods_pro_v2_HD.txt
```

### Linux
```
nano output/videos/airpods_pro_v2_HD.txt
```

## ❓ 자주 묻는 질문

### Q1: 설명란이 너무 길어요
**A**: 긴 버전(v3)은 상세하고, 짧은 버전(v1)은 간결합니다. 원하는 버전을 선택하세요!

### Q2: 해시태그를 바꾸고 싶어요
**A**: `.txt` 파일에서 해시태그 부분만 수정하면 됩니다.

### Q3: 언어를 바꿀 수 있나요?
**A**: 생성 시 `--lang` 옵션으로 언어를 선택하면 자동으로 해당 언어로 생성됩니다.

### Q4: 설명란만 다시 생성할 수 있나요?
**A**: 현재는 전체 재생성이 필요합니다. 하지만 `.txt` 파일을 직접 수정하는 것이 더 빠릅니다!
