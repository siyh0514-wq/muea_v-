# 🌐 웹 기반 키워드 선택 시스템

## 개요

**클릭만으로** 키워드와 제목을 선택할 수 있는 웹 인터페이스입니다.

터미널에서 번호를 입력하는 대신, 브라우저에서 키워드와 제목을 **직접 클릭**하여 선택할 수 있습니다.

## 🚀 사용 방법

### 방법 1: 메인 스크립트 실행

```bash
# 웹 UI 모드로 실행
python main.py --web
```

### 방법 2: 직접 실행

```bash
# 웹 서버 직접 실행
python keyword_selector_web.py
```

실행하면:
1. 자동으로 브라우저가 열립니다
2. `http://localhost:5000` 접속
3. 웹 인터페이스에서 작업

## 📱 사용 흐름

### 1. 주제 입력
![주제 입력](../docs/images/web-ui-step1.png)

- 입력창에 주제 입력 (예: 청년도약계좌)
- "분석 시작" 버튼 클릭

### 2. AI 분석
- 로딩 화면이 표시됩니다
- Gemini AI가 키워드와 제목을 분석합니다

### 3. 키워드 선택 (클릭!)
![키워드 선택](../docs/images/web-ui-step2.png)

- **고수익 키워드 카드**를 클릭하여 선택
- 여러 개 선택 가능
- 선택된 카드는 파란색으로 표시됩니다

각 카드에는 다음 정보가 표시됩니다:
- 키워드명
- 유형 (행동유도형, 고연령타겟형, 금융/돈관련)
- 경쟁도 (low, medium, high)
- CPC 잠재력 (높음, 중간, 낮음)

### 4. 제목 선택 (클릭!)
![제목 선택](../docs/images/web-ui-step3.png)

- **추천 제목 카드**를 클릭하여 선택
- 1개만 선택 가능
- CTR 점수가 높은 제목이 표시됩니다

각 카드에는 다음 정보가 표시됩니다:
- 제목
- 훅킹 요소
- CTR 점수 (85-90/100)

### 5. 선택 확인
- 선택한 키워드와 제목이 하단에 요약 표시됩니다
- "🎬 YouTube 숏폼 스크립트 자동 생성" 버튼 클릭

### 6. 완료
![완료](../docs/images/web-ui-step4.png)

- 스크립트가 자동 생성됩니다
- 저장 위치와 다음 단계가 표시됩니다

## ✨ 주요 기능

### 1. 클릭 기반 선택
- **마우스 클릭**만으로 키워드/제목 선택
- 번호 입력 불필요
- 시각적으로 선택 상태 확인

### 2. 다중 선택
- 키워드: 여러 개 선택 가능
- 제목: 1개만 선택

### 3. 실시간 업데이트
- 선택할 때마다 실시간으로 요약 업데이트
- 선택 취소도 클릭 한 번

### 4. 반응형 디자인
- 데스크톱, 태블릿, 모바일 모두 지원
- 화면 크기에 자동 조정

### 5. 시각적 피드백
- 선택된 항목: 파란색 강조
- 호버 효과: 마우스 올리면 미리보기
- CTR 점수: 명확한 배지 표시

## 💡 장점

### vs 터미널 버전

| 특징 | 터미널 | 웹 UI |
|------|--------|-------|
| 선택 방식 | 번호 입력 | 클릭 |
| 시각적 | ✗ | ✅ |
| 다중 선택 | 쉼표로 구분 | 클릭만 |
| 미리보기 | ✗ | ✅ |
| 모바일 | ✗ | ✅ |
| 접근성 | 개발자 친화적 | 일반 사용자 친화적 |

### 주요 이점
1. ✅ **직관적**: 클릭만으로 선택
2. ✅ **시각적**: 모든 정보가 한눈에
3. ✅ **편리함**: 번호 기억 불필요
4. ✅ **오류 방지**: 잘못된 입력 불가능
5. ✅ **빠른 선택**: 마우스 클릭이 타이핑보다 빠름

## 🔧 설정

### 필수 패키지 설치

```bash
pip install flask
# 또는
pip install -r requirements.txt
```

### 포트 변경

기본 포트: 5000

다른 포트를 사용하려면:

```python
# keyword_selector_web.py 수정
run_web_ui(port=8080)  # 원하는 포트 번호
```

### API 키 설정

`.env` 파일에 Gemini API 키 필요:

```bash
GEMINI_API_KEY=your-key-here
```

API 키가 없어도 **기본 키워드**로 작동합니다.

## 🔍 문제 해결

### 브라우저가 자동으로 열리지 않는 경우
수동으로 브라우저를 열고 `http://localhost:5000` 접속

### 포트가 이미 사용 중인 경우
```bash
# 다른 포트 사용
python keyword_selector_web.py --port 8080
```

### Flask가 설치되지 않은 경우
```bash
pip install flask
```

### 스타일이 적용되지 않는 경우
브라우저 캐시 삭제 또는 강력 새로고침 (Ctrl+F5)
