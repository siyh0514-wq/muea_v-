## 🎯 실제 사용법

### 균형형 설정 (.env 파일)

```bash
# 이미지 분석용 (빠르고 저렴)
GEMINI_API_KEY=your_gemini_key

# 대본 생성용 (고품질)
OPENAI_API_KEY=your_gpt_key

# 비디오 생성용 (필수)
DID_API_KEY=your_did_key

# 하이브리드 모드 활성화
USE_HYBRID=true
```

### 명령어

```bash
# 하이브리드 모드로 실행
python auto_video_creator.py --lang ko --hybrid

# 또는 자동 (설정 파일에서 읽음)
python auto_video_creator.py --lang ko
```

## 💡 각 API 장점 정리

### Gemini의 강점
✅ **이미지 분석**: 제품 인식 빠르고 정확
✅ **비용**: 거의 무료 (60개/월 무료)
✅ **속도**: 매우 빠름 (1-2초)
✅ **무료 티어**: 테스트에 최적

### GPT-4o의 강점
✅ **텍스트 품질**: 가장 자연스러운 대본
✅ **설득력**: 구매 유도 문구 탁월
✅ **창의성**: 독특한 표현, 캐치프레이즈
✅ **SEO**: 키워드 최적화 능력

### D-ID의 강점
✅ **립싱크**: 자연스러운 입 움직임
✅ **음성**: 5개 언어 완벽 지원
✅ **화질**: 1080p/4K 고화질
✅ **속도**: 3-5분 빠른 생성
