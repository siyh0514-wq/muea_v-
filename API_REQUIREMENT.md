# 🔑 필요한 API

## 필수 API (2개만!)

### 1. Gemini AI (무료 시작 가능)
- **발급**: https://makersuite.google.com/app/apikey
- **비용**: 
  - 무료: 60개 요청/월
  - 유료: $0.001/요청 (초저렴!)
- **용도**: 이미지 분석, 키워드 추출, 대본 생성

### 2. D-ID (비디오 생성)
- **발급**: https://studio.d-id.com/account-settings
- **비용**: $0.08/영상 (20초 기준)
- **용도**: 정적 이미지를 말하는 비디오로 변환

---

## 총 비용

**영상 1개당**: 약 $0.08 (Gemini 거의 무료)

**월 100개 영상**: 약 $8

**vs 수동 제작**: $2,000-5,000

**절감율**: 99.6%

---

## API 키 설정

`.env` 파일 생성:
```
GEMINI_API_KEY=여기에_제미나이_키_입력
DID_API_KEY=여기에_D-ID_키_입력
```

끝!
